# Simple-dungeon-generator

Another small algorithm for your roguelike game level generation.

### Sample usage:
```
#include"Dungeon.h"

int main()
{
	Dungeon x(64, 64, 5);
	x.gen(60);
	x.show();

	std::cout << "Enter to end... ";
	std::cin.get();

	return 0;
}
```
